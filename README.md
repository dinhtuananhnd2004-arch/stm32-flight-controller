# STM32 Flight Controller

## Overview
This project is a flight controller designed using STM32 for UAV applications.

## Features
- PWM control for ESC
- IMU sensor integration (Gyroscope + Accelerometer)
- Real-time stabilization algorithm

## Hardware
- STM32 microcontroller
- IMU sensor (MPU6050 / MPU9250)
- Custom PCB design

## Tools
- Altium Designer
- STM32CubeIDE

## Results
- Stable motor control
- Successful testing on UAV platform
